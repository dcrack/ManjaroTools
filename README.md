# ManjaroTools
个人在使用Manjaro Linux时写的一些实用小脚本

使用方法：

        wget https://raw.githubusercontent.com/dcrack/ManjaroTools/master/xk.sh
        chmod 777 xk.sh
        ./xk.sh
        
        wget https://raw.githubusercontent.com/dcrack/ManjaroTools/master/ld.sh
        chmod 777 ld.sh
        ./ld.sh
